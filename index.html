# Date Time Value Meal

This tool supports the TaiDoc TD-4235 glucose meter and its OEM variants, including Méry, DiCare, Fora, and GlucoSure models using the CP2110 USB HID bridge. All processing happens locally in your browser - no data is uploaded or stored.

This project is community-maintained and not affiliated with TaiDoc or any OEM brand. Always consult your healthcare provider for medical decisions.
